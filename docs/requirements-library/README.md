# Requirements Library

本目录用于归档 AI Agent Platform 项目的需求文档、技术文档原始版本与后续整理版本。

## 当前归档

- `requirements.original.md`：原始需求文档归档
- `technical-design.original.md`：原始技术设计文档归档

## 后续建议

- 在本目录新增 `requirements.reviewed.md` 存放审核后的需求文档
- 在本目录新增 `technical-design.updated.md` 存放根据最新需求修订后的技术文档
- 对每次重要修订增加版本号与日期说明
